# Project Alpha: Food Additive & Health Impact Tracker

Project Alpha is a functional web application designed to provide a "third-eye perspective" on food additives. It moves beyond corporate marketing and regulatory labels to reveal the systemic health and societal impacts of common industrial ingredients.

## 🚀 Motivation
In a globalized food system, additives are often marketed as "safe" or "necessary" for shelf-life and texture. Project Alpha aims to unmask these ingredients, highlighting their roles in metabolic dysfunction, neurological impact, and the broader industrialization of our diet.

- **F#** - Core functional logic

- **WebSharper UI.Next** - Reactive UI components
- **Tailwind CSS** - Modern styling
- **GitHub Pages** - Hosting and deployment

## 🏗 Build & Run
### Prerequisites
- .NET 8.0 SDK
- Node.js 20

### Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/3lytg/project-alpha.git
   ```
2. Navigate to the project directory:
   ```bash
   cd project-alpha
   ```
3. Install npm dependencies:
   ```bash
   npm install
   ```
4. Build the project:
   ```bash
   dotnet build
   ```
5. Bundle the JavaScript:
   ```bash
   node esbuild.config.mjs
   ```
